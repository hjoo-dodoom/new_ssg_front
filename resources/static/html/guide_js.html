체크 전체 선택
전체 선택 그룹을 .label-control로 묶고 전체 선택할 input[type="checkbox"]에 check-all 클래스 붙여줍니다. 
					
//common.js
<div class="label_control">
    <label class="checkbox checkbox_type2">
        <input type="checkbox" class="check_all">
        <span class="ck_icon"></span>
        전체
    </label>
    <label class="checkbox checkbox_type2">
        <input type="checkbox">
        <span class="ck_icon"></span>
        체크1
    </label>
    <label class="checkbox checkbox_type2">
        <input type="checkbox">
        <span class="ck_icon"></span>
        체크2
    </label>
    <label class="checkbox checkbox_type2">
        <input type="checkbox">
        <span class="ck_icon"></span>
        체크3
    </label>
</div>
체크 전체 선택 위에 또 전체가 있는경우
상위 그룹 element에 label-control-parent 클래스를 추가로 붙여주고 상위 그룹 전체 체크에 check-all-parent를 추가로 붙여줍니다.
