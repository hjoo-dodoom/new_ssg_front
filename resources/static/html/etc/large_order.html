주문 신청 시 안내사항

1. 주문 파일 등록 시 엑셀파일에 작성한 내용 중 잘못 작성한 경우 화면에서 수정할 수 있습니다

2. 반드시 [주문신청]버튼을 클릭 하셔야 주문이 완료됩니다

3. 승인이 완료된 건은 [배송준비중]으로 표시되며, 우측 하단의 [주문/배송 조회]를 선택하시면 확인하실 수 있습니다
